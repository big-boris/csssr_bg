# csssr_template

После клонирования репозитория необходимо сделать
```sh
$ bower i
$ npm i
```
Потом останется только запустить команду
```sh
$ gulp
```
Для открытия проекта в браузере - ввести в адресной строке
```sh
http://localhost:8080/
```


---
# Используемые модули

 - gulp
 - gulp-autoprefixer
 - gulp-concat
 - gulp-csso
 - gulp-jade
 - gulp-stylus
 - gulp-uglify
